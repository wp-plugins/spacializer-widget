Spacializer - A WordPress Widget - User Controlled SoundMixing and Lightshow Widget 

== Description ==

The Spacializer Widget represents multiple sound samples, each as a luminescent orb of color. The volume of each sample varies depending upon how close your mouse pointer is to the associated colored orb. The widget height and width are configurable via Wordpress' widget admin panel. 

== Installation ==

Upload the folder "spacializer" into your "wp-content/plugins" directory.
Log in to Wordpress Administration area, choose "Plugins" from the main menu, find "Spacializer" 
and click the "Activate" button. 

--Configuration--
From the main menu choose "Design->Widgets."   
Drag the "Spacializer" widget to the preferred position on your sidebar.
Click the "Change" button below the widget to display the configuration menu.
Complete the configuration menu: 

	1. Widget Height - Specify the height in pixels (ex. 250px).  Be sure to include 'px' 
	2. Widget Width  - Specify the width in pixels (ex. 250px).  Be sure to include 'px' 

Click 'Save Changes' located below the configuration menu to save your changes.

--Uninstallation--
Log in to Wordpress Administration area.
Choose "Plugins" from the main menu
Find the name of the plugin "Spacializer", and click the "Deactivate" button.
If you were using the widget it will no longer appear, so be certain to review your site.
